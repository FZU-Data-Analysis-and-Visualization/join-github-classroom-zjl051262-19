# 数据分析常用 Markdown 语法总结

Markdown 是数据分析报告中常用的轻量级写作语法，可以把文字说明、表格、代码和公式组织在同一个文档中。它适合记录分析过程、展示分析结果，并提升报告的可读性。

## 2. 列表与表格

列表适合梳理数据分析流程和工具清单，表格适合展示结构化数据。两者结合使用，可以让报告内容更紧凑、信息更清晰。

### 有序列表：数据分析常用工具

1. Python：适合进行数据清洗、统计分析、建模和自动化处理。
2. Excel：适合完成基础数据整理、透视表分析和简单图表制作。
3. Tableau：适合制作交互式仪表板和商业数据可视化。

### 无序列表：Markdown 在数据分析中的核心用途

- 整理数据预处理步骤。

  - 柱状图说明：适合比较不同类别、地区或产品的指标差异。

## 3. 代码与公式

代码块可以记录数据处理过程，数学公式可以表达统计指标的计算方法。二者常用于数据分析报告的方法说明部分。

```python
import pandas as pd

# 读取销售数据
data = pd.read_csv("sales_data.csv")

# 计算销售额均值
avg_sales = data["销售额"].mean()
print(f"平均销售额：{avg_sales:.2f}万元")
```

代码功能：读取销售数据并计算平均销售额，适合用于快速检查月度销售表现。

均值公式：

$$
\bar{x} = \frac{1}{n}\sum_{i=1}^{n}x_i
$$

方差公式：

$$
\sigma^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2
$$

公式说明：均值用于描述一组数据的平均水平，方差用于衡量数据围绕均值的离散程度。

## 4. 综合实践报告：学生成绩数据分析

### 数据概况

本报告以某班级学生成绩为分析对象，数据包含语文、数学、英语三门课程成绩。分析目标是了解学生整体表现，并找出需要重点关注的学科。

### 分析方法

- 计算每门课程的平均分。
- 比较不同学科的成绩差异。
- 观察低分科目，辅助后续教学改进。

### 成绩汇总表

| 科目 | 平均分 | 最高分 | 最低分 |
| :--- | ---: | ---: | ---: |
| 语文 | 84 | 96 | 68 |
| 数学 | 79 | 98 | 55 |
| 英语 | 86 | 97 | 72 |

从表格可以看出，**英语平均分最高**，数学平均分相对较低，说明数学可能是后续学习辅导的重点方向。

```python
scores = {
    "语文": [84, 96, 68],
    "数学": [79, 98, 55],
    "英语": [86, 97, 72],
}

for subject, values in scores.items():
    print(subject, "平均分：", sum(values) / len(values))
```

综合结论：本次成绩分析显示班级整体成绩较稳定，但数学成绩差异较明显。后续可以结合错题类型和课堂表现进一步分析，以制定更有针对性的提升方案。
